> Tsrv is based on rollup, 0 Zero-config can quickly develop vue 3 component library

### 🏠 [Homepage](https://github.com/jaskang/tsrv)

## Install

```sh
yarn add tsrv -D
```

## Usage

```json
{
  "scripts": {
    "dev": "tsrv dev",
    "build": "tsrv build",
    "test": "tsrv test"
  }
}
```
